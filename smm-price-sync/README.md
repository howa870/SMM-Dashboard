# SMM Price Sync System

نظام تلقائي لمراقبة أسعار خدمات SMM Panel وتحديثها مع إشعارات Telegram.

## الميزات

- **جلب الخدمات** من API المزود (Followiz أو أي SMM API)
- **تسعير تلقائي** بهامش ربح 50% قابل للتخصيص
- **تحديث تلقائي** كل 5 دقائق — يرفع السعر فقط، لا يخفضه أبداً
- **إشعارات Telegram** فورية عند ارتفاع أي سعر
- **حماية من الأخطاء** — إعادة محاولة تلقائية، النظام لا يتوقف
- **تخزين JSON** لجميع الأسعار والتغييرات

## هيكلة المشروع

```
smm-price-sync/
├── index.js       # نقطة البداية والجدولة
├── services.js    # جلب الخدمات من API
├── pricing.js     # حساب الأسعار وهامش الربح
├── updater.js     # منطق الفحص والتحديث
├── telegram.js    # إرسال إشعارات Telegram
├── data/
│   └── prices.json  # قاعدة الأسعار (تُنشأ تلقائياً)
├── .env           # المتغيرات السرية (أنشئه من .env.example)
├── .env.example   # مثال على المتغيرات المطلوبة
└── package.json
```

## طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
cd smm-price-sync
npm install
```

### 2. إعداد المتغيرات
```bash
cp .env.example .env
```
ثم عدّل `.env`:
```env
API_KEY=your_followiz_api_key
API_URL=https://followiz.com/api/v2
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
MARKUP=1.5
```

### 3. الحصول على CHAT_ID
1. أرسل أي رسالة لبوتك على Telegram
2. افتح: `https://api.telegram.org/bot<TOKEN>/getUpdates`
3. انسخ قيمة `"id"` من قسم `"chat"`

### 4. تشغيل النظام
```bash
npm start
```

## كيف يعمل

```
كل 5 دقائق:
  ┌─────────────────────────────────────────┐
  │  1. جلب الخدمات من API                  │
  │  2. لكل خدمة:                           │
  │     - إذا كانت جديدة → حفظها            │
  │     - إذا ارتفع السعر → تحديث + إشعار  │
  │     - إذا انخفض السعر → تجاهل (لا تخفض)│
  │  3. حفظ القاعدة                         │
  └─────────────────────────────────────────┘
```
